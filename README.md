
# Expense Tracker

A simple Flask application for tracking expenses.


## Deployment

Clone or fork the repository

Go to Settings > Secrets and Variables > New Repository Secret

Add the following secret Variables

- DOCKER_USERNAME
- DOCKER_TOKEN
- POSTGRES_USER
- POSTGRES_PASSWORD
- SSH_PRIVATE_KEY
